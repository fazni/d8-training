Creating a Drupal 8 module

Pieter Frenssen / @pfrenssen

Stuff you need

  • IDE or text editor
  • LAMP stack
  • Composer

Starting a D8 project

https://github.com/drupal-composer/drupal-project


     $ composer create-project drupal-composer/drupal-project \
       --stability=dev mydir

Sanity check

  • Open Drupal in the browser

          $ drush st
          $ drupal

D8 developer mode

  • Display error messages
  • Disable CSS/JS aggregation
  • Disable render cache
  • Allow installing test modules

Engage developer mode

Copy:

sites/example.settings.local.php

To:

sites/default/settings.local.php

sites/default/settings.php


# if (file_exists(__DIR__ . '/settings.local.php')) {
#   include __DIR__ . '/settings.local.php';
# }

Uncomment these lines

More development options

sites/default/services.yml

  • Twig debugging
  • Twig auto-recompile
  • Disable Twig cache
  • Advanced caching options
  • ...

Let's get started

Fire up your consoles!

Generate module skeleton


            $ drupal generate:module
            Enter the new module name:
            > One

Generated files

  • one.info.yml
  • one.module
  • composer.json

Enable it!

Go to "Manage > Extend"

Create a form


            $ drupal generate:form

            Module name: one
            Form class: OneSubscriptionForm
            Type: email
            Input label: Your e-mail
            Machine name: email
            Description: Subscribe to our newsletter.

Generate a block


            $ drupal generate:plugin:block

            Module name: one
            Form class: OneSubscriptionBlock
            Theme region: sidebar_first
            Form structure: no

Generated code

src/Plugin/Block/OneSubscriptionBlock.php


/**
 * @Block(
 *  id = "one_subscription_block",
 *  admin_label = @Translation("One subscription block"),
 * )
 */
class OneSubscriptionBlock extends BlockBase {

  public function build() {
    $build = [];
    $build['one_subscription_block']['#markup'] = 'Implement OneSubscriptionBlock.';

    return $build;
  }

Show our form in the block


  public function build() {
    return \Drupal::formBuilder()->getForm(
      'Drupal\one\Form\OneSubscriptionForm'
    );
  }

See SearchBlock::build()

Add a submit button


  public function buildForm(array $form, FormStateInterface $form_state) {
    $form['email'] = ...

    $form['submit'] = array(
      '#type' => 'submit',
      '#value' => $this->t('Submit'),
    );

    return $form;
  }

Add a submit handler


  public function submitForm(array &$form, FormStateInterface $form_state) {
    // @todo Save it to the database.
    drupal_set_message($this->t('Thanks for subscribing.'));
  }

Working with config

List all config


            $ drush config-list (cli)

Show config


            $ drush cget one.oneconfiguration

Export config


            $ cd modules/custom/one
            $ mkdir -p config/install
            $ drush cget one.oneconfiguration >
                config/install/one.oneconfiguration.yml
